Portal central de presentaciones y servicios de Salesforce para Cirion Technologies.

```
├── index.html (landing page)
├── advisory-2026/
│   ├── 2026-07-07-account-plan-health-check/
│   └── 2026-07-13-plan-ejecucion-consolidado/
└── advisory-2027/
    └── (futuras presentaciones)
```

## Agregar Nueva Presentación

1. Crear carpeta: `advisory-YYYY/YYYY-MM-DD-nombre-descriptivo/`
2. Copiar HTML como `index.html` dentro de la carpeta
3. Actualizar `index.html` principal con la nueva entrada
4. Commit y push a GitHub
